Tab Bar Rules

  • Border-bottom on tab bar: 2px solid #e8e8e8
  • Active tab underline: 2.5px solid brand color, margin-bottom: -2px
  • Only ONE tab active at a time
  • Badge (red pill) on tabs that have pending items
  • Tab labels: sentence case, concise
  • Actions column changes per tab (e.g., no Approve/Reject on Disbursed tab)

Table Rules

  • border-collapse: collapse on table element
  • Rows alternate #EBF4FB / #fff (first data row = #EBF4FB)
  • Record IDs are blue and clickable (link to detail view)
  • Amounts bold (#111, font-weight 600)
  • Actions: View | Approve | Reject for pending tab
  • Separator | in #aaa between each action
  • Adapt column headers to the client's domain

Footer

  • "Showing N of N records" — left aligned, #aaa
  • ZoTok brand — right aligned, #aaa
  • Separated by border-top: 1px solid #f0f0f0, padding-top: 14px

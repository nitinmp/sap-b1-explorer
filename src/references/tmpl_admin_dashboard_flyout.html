Admin Dashboard Flyout

Right-side panel opened on "View" click · Overlays the table

When to Use

When the manager clicks View on any table row, a flyout panel slides in from the right showing the full record detail. The table remains visible but dimmed. Approve / Reject actions are in the flyout footer.

Section Structure

  • Each section: white card, radius 10px, border #e8e8e8
  • Section head: title (bold) + subtitle (grey), border-bottom
  • Fields: 2-column grid for short values, full-width for long text
  • Field label: uppercase, 11px, #888, letter-spacing .3px
  • Field value: 13px, #111, font-weight 500
  • Documents: icon + name/size + View link per row
  • Sections are scrollable — body has overflow-y: auto
  • Footer is sticky — never scrolls away

Footer Button Rules

  • Both buttons flex:1 (equal width)
  • border-radius: 8px, padding: 11px
  • Reject: white bg, red border + red text + X icon
  • Approve: blue bg (#2196F3) + white text + check icon
  • Adapt labels per use case (e.g., "Disburse", "Send Back")
